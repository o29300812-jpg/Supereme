# Supereme - Professional Legal Platform

## Overview
A comprehensive legal platform for law students and practitioners featuring:
- Law Books Library
- Urdu Law Dictionary with Google Integration
- Legal Cases Database
- Judgments Repository
- User Suggestions System
- Complete Admin Panel

## Features
- User Authentication (Sign Up/Sign In)
- Admin Dashboard with Full Control
- Modern Microsoft-inspired UI with Animations
- Responsive Design (RTL Support for Urdu)
- Google Search Integration
- Direct PDF Downloads
- Search History & Bookmarks
- User Ratings & Reviews

## Installation
1. Import database.sql into MySQL
2. Update Database.php credentials
3. Run on XAMPP/LAMP stack
